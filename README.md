## 项目简介

面向肇庆本地柑橘（砂糖橘/贡柑）的机器视觉品质检测系统，采用"传统 CV 分级 + YOLOv8 缺陷检测"双引擎方案，支持单张/批量图像的品质分级（一级/二级/等外品）与表面缺陷检测（病斑/裂纹/碰伤/畸形），提供溯源信息与市场价格参考。

## 核心功能

- ✅ 单张/批量图像上传与检测（PC Web + 移动端 H5）
- ✅ 品质分级：基于果径、着色率、缺陷占比的量化分级
- ✅ 缺陷检测：YOLOv8 定位病斑/裂纹/碰伤/畸形
- ✅ 结果可视化：标注图、等级徽章、量化指标、统计图表
- ✅ 溯源信息：产地/批次登记、二维码生成
- ✅ 市场价格参考
- ✅ 报表导出：PDF/Excel

## 技术栈

**前端**：Vue 3 + Vite + Element Plus + ECharts  
**后端 API**：Python 3.10 + FastAPI + SQLAlchemy + MySQL  
**算法服务**：OpenCV + PyTorch + YOLOv8 + scikit-learn  
**部署**：Docker Compose / 本地一键启动

## 快速开始

### 环境要求

- Python 3.10+
- Node.js 18+
- MySQL 8.0+
- Git

### 开发环境搭建

#### 1. 克隆仓库

```bash
git clone <repository-url>
cd citrus-vision
```

#### 2. 后端设置

```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

#### 3. 前端设置

```bash
cd frontend
npm install
npm run dev
```

#### 4. 数据库初始化

```bash
mysql -u root -p < database/schema.sql
mysql -u root -p < database/seed.sql
```

#### 5. 算法服务启动

```bash
cd algorithm-service
pip install -r requirements.txt
python server.py
```

### Docker Compose 一键启动（推荐）

```bash
docker-compose up -d
```

访问：http://localhost:8080
