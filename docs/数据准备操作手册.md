## 4. 标注规范

### 4.1 缺陷类别定义

| 类别 ID | 类别名称 | 英文标识 | 标注要求 |
|---|---|---|---|
| 0 | 病斑 | black_spot | 黑褐色斑点，边界清晰，包括黑星病、溃疡病 |
| 1 | 裂纹 | crack | 果皮开裂，有明显裂缝，包括顶裂、侧裂 |
| 2 | 碰伤 | bruise | 机械损伤导致的软化/变色/凹陷 |
| 3 | 畸形 | deformity | 果形不规则，明显偏离球形（扁平/凸起/弯曲） |

### 4.2 标注原则

1. **矩形框**：尽量紧贴缺陷边界，不包含过多正常区域
2. **重叠处理**：多个缺陷重叠时，分别框选（允许框重叠）
3. **边缘缺陷**：部分缺陷在图像边缘外，只框选可见部分
4. **模糊判断**：
   - 轻微色斑（不确定是否病斑）→ 不标注
   - 疑似裂纹（可能是果蒂痕迹）→ 标注为裂纹，复核时裁定
5. **畸形标注**：仅标注**严重畸形**（明显影响品质的），轻微不规则不标

### 4.3 YOLO 格式说明

每张图片对应一个 `.txt` 文件，每行一个缺陷：
```
<class_id> <x_center> <y_center> <width> <height>
```

- `class_id`：0~3（对应上述 4 类）
- 坐标：归一化到 [0, 1]，即除以图像宽高
- 示例：`0 0.512 0.348 0.156 0.203`（病斑，位于图像中央偏上）

**Roboflow 会自动生成此格式，无需手工计算。**

## 6. 数据划分与防泄露

### 6.1 数据集划分脚本

```python
# scripts/split_dataset.py
import os
import random
from pathlib import Path
from collections import defaultdict

def split_dataset(image_dir, output_dir, train_ratio=0.7, val_ratio=0.2, test_ratio=0.1):
    """
    按 7:2:1 划分数据集，分层抽样保证各类均衡
    防泄露：同一批次/同一果实的多角度图片划入同一子集
    """
    # 1. 按文件名前缀分组（假设同一果实命名为 xxx_001.jpg, xxx_002.jpg）
    groups = defaultdict(list)
    for img in Path(image_dir).glob("*.jpg"):
        # 提取前缀（去掉最后的 _数字）
        prefix = "_".join(img.stem.split("_")[:-1]) if "_" in img.stem else img.stem
        groups[prefix].append(img)
    
    # 2. 打乱分组并划分
    group_list = list(groups.values())
    random.shuffle(group_list)
    
    n_total = len(group_list)
    n_train = int(n_total * train_ratio)
    n_val = int(n_total * val_ratio)
    
    train_groups = group_list[:n_train]
    val_groups = group_list[n_train:n_train+n_val]
    test_groups = group_list[n_train+n_val:]
    
    # 3. 复制文件到对应目录
    for subset, groups in [("train", train_groups), ("val", val_groups), ("test", test_groups)]:
        subset_dir = Path(output_dir) / subset
        (subset_dir / "images").mkdir(parents=True, exist_ok=True)
        (subset_dir / "labels").mkdir(parents=True, exist_ok=True)
        
        for group in groups:
            for img in group:
                # 复制图片
                (subset_dir / "images" / img.name).write_bytes(img.read_bytes())
                # 复制标注
                label = img.with_suffix(".txt")
                if label.exists():
                    (subset_dir / "labels" / label.name).write_text(label.read_text())
    
    print(f"划分完成：train={len(train_groups)} groups, val={len(val_groups)}, test={len(test_groups)}")

if __name__ == "__main__":
    split_dataset("data/labeled", "data/split")
```
